## Requirements
- Python 3.9
- Numpy 1.21.5
- Pandas 1.4.1
- Prettytable 3.2.0
- Scipy 1.8.0
- torch 1.11.0+cu113

## How to use
- Different experiments can be performed by modifying the experiment parameters in ExperimentParams.py, but it is recommended to add loops in main.py to perform comparison experiments.
